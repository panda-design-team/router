## Get Started

- 安装

```bash
yarn add @panda-design/router
```

- 基于 `@panda-design/router` 构建应用

```typescript jsx
import {LeftNavigation} from '@panda-design/router';

const App = () => {
    return <LeftNavigation items={[/* ... */]} />;
};
```
